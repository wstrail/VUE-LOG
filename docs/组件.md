# 组件

## 一、创建项目

```npm
npm init vue@latest
npm install
npm run dev
```

## 二、组件组成

### 单文件组件（sfc）

```vue
<template>
    <div  class="">{{ message }}</div>
</template>
<script>
    export default {
        data(){
            return{
                message:"组件基础组成"
            }
        }
    };
</script>
<style scoped><style><!--scoped：让当前样式只在当前组件中生效-->
```

### 引用组件

```vue
<template>
    <name/>
</template>
<script>
    import "name"from"route"
    export default{
    compents:{
        name
    }
}
</script>
<style><style>
```
